# Ferretería El Tornillo Feliz — Calculadora de ventas

La ferretería usa este pequeño script para calcular el total de una venta en caja.
Un desarrollador junior escribió una primera versión, pero desde hace unas semanas
algunos clientes reclaman que el total en pantalla "no cuadra" cuando compran
grandes cantidades de un mismo producto.

Por separado, la dueña del negocio pidió que el sistema empiece a calcular el
Impuesto al Valor Agregado (IVA, 13%) sobre el monto de la venta, algo que el
script todavía no hace.

## Cómo correrlo

```bash
npm test
```

Vas a ver que algunas pruebas fallan. Ese es el punto de partida.
